# 🎆 Website Bán Đồ Tết - Spring Boot

Dự án assignment Java 5 - Website bán đồ Tết sử dụng **Spring Boot** + **Thymeleaf** + **JPA** + **SQL Server**

## 🚀 CÁCH CHẠY DỰ ÁN

### Yêu cầu:
- **Java 17+**
- **Maven** 
- **SQL Server** 

### Chạy project:
```bash
mvn spring-boot:run
```

### Truy cập:
- Website: http://localhost:8080
- Database: DoTetDB (SQL Server)
